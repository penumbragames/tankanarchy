# Tank Anarchy

Tank Anarchy is an online multiplayer tank battle game.

# Building This Project:

I am developing this project on node v20.19.5. This project uses
[`bun`](https://bun.com/) and [`justfile`](https://just.systems/man/en/) for
build/compilation.

```
npm install
just build
just start
```
